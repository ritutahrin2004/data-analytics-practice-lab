# A/B Testing Analysis: Website Design Impact

## Business Problem
An e-commerce company wants to test whether changing the website background 
from White (Group A) to Black (Group B) improves user conversion and engagement.

## Objectives
- Compare conversion rate between Group A and Group B  
- Analyze engagement using time spent and page views  
- Evaluate performance across device and location  

## Key Insights
- Conversion rate increased significantly (5.39% → 14.07%) in Group B  
- Time spent showed no meaningful difference between groups  
- Page views slightly decreased in Group B  
- Conversion improvement was consistent across devices and locations  

## Recommendations
- The new design shows strong potential to improve conversion  
- However, due to limited statistical validation, results should be treated as directional  
- A controlled experiment is recommended before full rollout  

## Tools Used
- Python (Pandas, NumPy)  
- Basic statistical testing  

## Note
This analysis is based on a synthetic dataset and is intended for practice purposes.
